手順0.
　インフラチュートリアルの手順１と同じ動作でターミナルを開きます。
　（すでに開いている場合はそのままで大丈夫です）

手順1.
　以下のコマンドを入力します。
　　「yum -y install httpd」
　yumは平たく言えばソフトウェアをインストールするコマンドです。
　　→OSや環境によって使えるコマンドが異なります。
　-yオプションは全ての問いかけにyesで答えるという意味です。
　（容量大きいけど大丈夫ですか？とか、既にインストールされてますが上書きしますか？とか）
　installはそのままインストールするというコマンドです。
　httpdはサーバとして機能するために必要なソフトウェアです。

　つまり、webサーバのためのソフトをインストールしますという操作です。

手順2.
　「service httpd start」
　でhttpdを起動します。
　※serviceはcentos6でのコマンドなので、OS等によっては異なります。
　※startの他に、restart(再起動)、stop（停止）などのコマンドがあります。

手順3.
　「service httpd status」
　で現在のhttpdの状態を確認します。
　「running」となっていればOKなはずです。

手順4.
　下記コマンドでhttpdの自動起動をONにします。
　　「chkconfig httpd on」 

手順5.
　ブラウザでアクセスするために、IPアドレスを下記コマンドで確認します。
　　「ifconfig」
　基本的に「wlan0」の中に「192.168.xxx.yyy」（末尾が1は除く）があるので、それのはずです。
　※有線LANで接続している時はeth0の中にあります。

手順6.
　CentOSのfirefoxで、手順5.で調べたIPアドレスを入力します。
　Apacheのデフォルトページが出てきたら成功です。
